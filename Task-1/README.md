# Task 1: Rule-Based ChatBot

A simple chatbot that responds to user inputs based on predefined rules using if-else statements and pattern matching. This project demonstrates a basic understanding of natural language processing and conversation flow.

## How to Run

```bash
python chatbot.py
```

## Supported Commands

| Category | Keywords |
|----------|----------|
| Greetings | hello, hi, hey, good morning, good night |
| About Bot | your name, who are you, how are you |
| Time/Date | time, date, today |
| Entertainment | joke, fact, motivate, quote |
| Calculator | calculate 5 + 3 |
| Topics | music, movie, food, sport, game, python |
| Mood | happy, sad, bored |
| Help | help, what can you do, how can you help |
| Exit | bye, exit, quit |
