# Task 4: Universal Recommendation System

A simple, comment-free content-based recommendation system built with Python, Pandas, and Scikit-Learn. It recommends Movies, Books, or Products based on the similarity of their attributes using TF-IDF vectorization and Cosine Similarity.

## How to Run

```bash
python recommendation_system.py
```

## Features
- **Content-Based Filtering**: Suggests items similar to what the user likes.
- **TF-IDF & Cosine Similarity**: Uses text processing to find similarities between items.
- **Three Categories**: Supports recommending Movies, Books, and Products.
- **Interactive Menu**: Choose a category and type an item name to get top 3 recommendations.

## Requirements

Ensure you have the required libraries installed:
```bash
pip install pandas scikit-learn
```
